# The Heritage Trail

A walking trail mockup for **St Peters Lutheran College Indooroopilly**.

## Live site

Once GitHub Pages is enabled, this site will be available at:

**https://sarretechnologies.github.io/heritage-trail/**

## Repository contents

- `index.html` — the full self-contained mockup (HTML + inline CSS + Leaflet map via CDN)
- `README.md` — this file

## Local preview

The site is a single static HTML file. To preview locally, open `index.html` in any modern browser, or serve the folder with any static server, e.g.:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Notes

This is a design mockup. Map tiles and fonts load from public CDNs (Leaflet, Google Fonts), so an internet connection is required for full rendering.
